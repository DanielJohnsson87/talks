# 3.2 🤷‍♂️ A State change is started by dispatching an action
![Redux-2](Redux-2.png)


```javascript
import { configureStore } from "@reduxjs/toolkit";

const store = configureStore({
	reducer: (state, action) => state
});

store.dispatch({ 
	type: "VEHICLE_ADDED",
	payload: { id: 1, name: "Vehicle 1" }
})
```
